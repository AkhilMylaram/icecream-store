# **App Name**: Creamery Central

## Core Features:

- Product Catalog Aggregation: Aggregate product data from microservices to create a central product catalog.
- Order Management: Manage orders, dispatch them to the respective microservices for processing.
- User Authentication: Implement secure user authentication and authorization across all integrated microservices.
- Personalized Recommendations: Provide personalized ice cream recommendations based on user preferences using an AI tool.
- Real-time Inventory: Display real-time inventory information aggregated from the microservices.
- Promotions and Discounts: Create and display promotions and discounts managed across the connected services.

## Style Guidelines:

- Primary color: Saturated Raspberry (#D3476B), reflecting the sweetness and vibrancy of ice cream flavors.
- Background color: Pale Rose (#F8E5EB), providing a soft and inviting backdrop to showcase the products.
- Accent color: Sky Blue (#6BB4DA), contrasting with the raspberry tones to draw attention to key elements.
- Headline font: 'Playfair', a modern sans-serif with high contrast, lending an elegant and high-end feel. Note: currently only Google Fonts are supported.
- Body font: 'PT Sans', a humanist sans-serif font offering a balance of modern appeal and readability. Note: currently only Google Fonts are supported.
- Use custom, cheerful icons related to different ice cream flavors and ingredients.
- Employ a card-based layout to showcase ice cream products, integrating high-quality images and concise details.
- Implement subtle, engaging animations on product hover and interaction to enhance the user experience.
